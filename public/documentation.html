My Movie Database

This web application will provide you with information about different movies, directors, and genres.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET none A JSON object holding data about all the movies
Get data about a single movie by title /movies[title] GET none A JSON object holding data about a single movie, containing a title and director name.
Example:
{
Title: "Mama Mia!",
director: "Phyllida Lloyd"
}
Get data about a genre by name /genres/[name] GET none A JSON object holding data about a genre.
Get data about a director by name /directors/[name] GET none A JSON object holding data about a director.
Add a user /users POST A JSON object holding data about the user to add.
structrured like:
{
name: "John Doe",
email: "john.doe@example.com",
password: "testtest"
}
 A JSON object holding data about the user that was added:
{
name: "John Doe",
email: "john.doe@example.com",
password: "testtest"
}
Update user name /users[name] PUT none A text message indicating the user name that was updated
Add movie to favorite list of user /users/[name]/favoritelist POST none A text message indicating the movie that was added
Delete movie from favorite list of user /users/[name]/favoritelist DELETE none A text message indicating wether the movie was successfully removed from the favorite list
Delete user /users/[name] DELETE none A text message indicating wether the user was successfully removed
