My Movie Database

This web application will provide you with information about different movies, directors, and genres.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET none A JSON object holding data about all the movies
Get data about a single movie by title /movies[title] GET none A JSON object holding data about a single movie, containing a title and director name.
Example:
{
Title: "Mama Mia!",
Genre:
director: "Phyllida Lloyd"
}
Get data about all genres /genres GET none A JSON object holding data about all genres.
Get data about a genre by name /genres/[Name] GET none A JSON object holding data about a specific genre.
Get data of all directors /directors GET none A JSON object holding data about all directors.
Get data of a director by name /directors/[name] GET none A JSON object holding data about a specific director.
Get data of all users /users GET None A JSON object holding data about all the users
Get data of a user by name /users[Name] GET none A JSON object holding data about a single user.
Add a new user /users POST A JSON object holding data about the user to add.
structrured like:
{
name: "John Doe",
email: "john.doe@example.com",
password: "testtest"
birthday: "15/08/80"
}
 A JSON object holding data about the user that was added:
{
name: "John Doe",
email: "john.doe@example.com",
password: "testtest"
birthday: "15/08/80"
}
Add movie to favorite list of user /users/[Username]/movies/[MoviesID] POST none A JSON object holding data about the movie that was added.
Update a users username /users[Username] PUT none A text message indicating the user name that was updated
Delete movie from favorite list of user /users/[Username]/movies/[MovieID] DELETE none A text message indicating wether the movie was successfully removed from the favorite list
Allow users to deregister /users/[Username] DELETE none A text message indicating wether the user was successfully deregistered
